# Ibrahim Digital - Premium Digital Marketing Agency Website

A modern, highly-responsive digital marketing agency website built with React, Vite, and Tailwind CSS. The website features smooth scrolling, framer-motion animations, and a sleek dark navy & primary blue color palette.

## Features

- **Modern Tech Stack**: React, Vite, Tailwind CSS
- **Smooth Animations**: Framer Motion for scroll reveals and page transitions, Lenis for smooth scrolling
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Custom Components**: Includes an interactive Hero dashboard, Services layout, About page, and Contact forms
- **Icons**: Lucide React icons for lightweight, scalable graphics

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/amaanshaikh711/Digital-Agency-Website.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Digital-Agency-Website
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components/`: Reusable UI components (Navbar, Footer, HeroDashboard)
- `src/pages/`: Main page components (Home, About, Services, Contact, ServiceDetail)
- `src/utils/`: Utility functions (e.g., `cn` for Tailwind class merging)
- `src/index.css`: Global styles and Tailwind configuration
- `public/`: Static assets
